# "惠电"宿舍电费充值管理系统开发计划

## 一、项目概述
基于Qt 6.5.3的C/S架构宿舍电费充值管理系统，包含学生端和管理员端功能。

## 二、功能模块设计

### 1. 登录模块
- 用户身份认证（学生/管理员）
- 密码加密存储
- 登录状态管理

### 2. 数据库设计（SQLite）
- 用户表（学生/管理员）
- 宿舍信息表
- 电费记录表
- 充值记录表
- 电表数据表

### 3. 学生端功能
- 个人信息查看
- 宿舍电费余额查询
- 电费使用记录查询
- 在线充值功能
- 充值记录查询

### 4. 管理员端功能
- 学生信息管理
- 宿舍信息管理
- 电费数据录入
- 充值记录查询与统计
- 系统设置

## 三、实施步骤

### 第一阶段：数据库设计与基础架构
1. 创建数据库管理类（DatabaseManager）
2. 设计并创建数据库表结构
3. 实现数据库连接与初始化

### 第二阶段：登录模块完善
1. 完善登录界面
2. 实现用户认证逻辑
3. 添加权限管理

### 第三阶段：学生端功能开发
1. 学生主界面设计
2. 实现余额查询功能
3. 实现电费记录查询
4. 实现充值功能

### 第四阶段：管理员端功能开发
1. 管理员主界面设计
2. 实现学生信息管理
3. 实现宿舍信息管理
4. 实现电费数据管理
5. 实现统计报表功能

### 第五阶段：系统优化与测试
1. 界面美化与用户体验优化
2. 功能测试与bug修复
3. 编写系统设计文档
4. 编写测试报告

## 四、技术栈
- 开发框架：Qt 6.5.3
- 编程语言：C++17
- 数据库：SQLite
- UI设计：Qt Designer
