# 网页电费查询功能使用说明

## 功能概述

本系统提供了**网页电费自动解析功能**，可以直接从一卡通网站获取并解析电费数据！

## 功能特性

### 1. 直接获取网页数据
- 使用HTTP请求直接获取网页内容
- 智能解析HTML提取剩余电费度数
- 支持提取剩余金额和宿舍信息

### 2. 多种正则表达式匹配
- 自动识别多种表述方式（剩余、余额、度数等）
- 支持度、kWh、千瓦时等单位
- 容错性强，适应不同网页格式

### 3. 调试功能
- 显示原始HTML内容（前5000字符）
- 进度条显示获取状态
- 详细的错误提示

### 4. 备选方案
- 同时提供浏览器打开功能
- 可以先在浏览器登录再获取数据
- 查询信息记录功能保留

## 使用步骤

### 方法一：直接获取（推荐先试）

1. 登录学生端系统
2. 点击顶部的"网页电费查询"按钮
3. 确认网址正确（默认已填好）
4. 点击绿色的**"获取电费数据"**按钮
5. 等待数据获取完成
6. 查看解析结果

### 方法二：先登录再获取

1. 点击**"在浏览器中打开"**按钮
2. 在浏览器中完成一卡通登录
3. 导航到电费查询页面
4. 回到系统，点击**"获取电费数据"**
5. 查看解析结果

## 解析规则

系统会尝试匹配以下模式来提取电费数据：

### 剩余电费度数匹配
- `剩余 X 度`
- `剩余 X kWh`
- `剩余 X 千瓦时`
- `剩余电量: X`
- `剩余度数: X`
- `电量剩余: X`

### 剩余金额匹配
- `剩余 X 元`
- `余额: X`
- `剩余金额: X`

### 宿舍信息匹配
- `宿舍: XXX`
- `房间: XXX`
- `寝室: XXX`

## 常见问题

**Q: 点击"获取电费数据"没反应？**
A: 请检查网络连接，确保可以访问校园网。

**Q: 提示"需要登录"怎么办？**
A: 先点击"在浏览器中打开"，在浏览器中完成登录，再回来获取数据。

**Q: 解析不到剩余电费度数？**
A: 查看原始HTML区域，确认网页内容是否正确加载。

**Q: 可以自定义网址吗？**
A: 可以，在"目标网址"框中修改即可。

**Q: 数据会保存吗？**
A: 当前版本不会自动保存，请手动记录到系统中。

## 技术实现

### 核心类：ElectricityParser

```cpp
// 主要方法
void fetchElectricityData();           // 获取数据
void fetchElectricityData(url);        // 获取指定URL
QString getRemainingKwh();              // 获取剩余度数
QString getRemainingAmount();           // 获取剩余金额
QString getDormitory();                 // 获取宿舍
QString getRawHtml();                   // 获取原始HTML

// 信号
void dataReady();                       // 数据准备好
void errorOccurred(error);             // 发生错误
```

### 工作流程

1. 创建HTTP请求（含User-Agent）
2. 发送GET请求获取网页
3. 接收HTML响应
4. 使用正则表达式匹配数据
5. 提取并显示结果
6. 发出dataReady信号

## 注意事项

### 1. 登录问题
- 该网页可能需要先登录才能访问
- 建议先用浏览器打开并登录
- 登录后再尝试获取数据

### 2. 网络要求
- 需要校园网环境
- 确保能访问一卡通网站
- HTTPS证书会自动处理

### 3. 数据准确性
- 解析结果仅供参考
- 请以网站显示为准
- 可以查看原始HTML确认

### 4. 隐私安全
- 密码等信息不会被系统保存
- 仅在当前会话中使用
- 建议在安全网络环境使用
