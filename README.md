# 惠电宿舍电费充值管理系统

## 项目简介

"惠电"是一个基于Qt 6.5.3和C++17开发的C/S架构宿舍电费充值管理系统，旨在为高校提供便捷、高效的电费管理解决方案。

## 功能特性

### 学生端功能
- 用户登录与登出
- 个人信息查看
- 电费余额查询（余额不足50元时红色预警）
- 在线充值功能
- 电费使用记录查询
- 充值记录查询

### 管理员端功能
- 用户登录与登出
- 系统统计（学生总数、总余额、总充值金额）
- 学生信息管理（增删改查）
- 宿舍信息管理（增删改查）
- 为学生充值
- 电费扣费
- 充值记录查询
- 用电记录查询

## 技术栈

- **开发框架**: Qt 6.5.3
- **编程语言**: C++17
- **数据库**: SQLite
- **架构模式**: C/S（客户端/服务器）架构

## 快速开始

### 环境要求

- Qt 6.5.3 或更高版本
- MinGW 或 MSVC 编译器
- Windows 操作系统

### 编译步骤

1. 打开 Qt Creator
2. 打开 `GraduatingDesice.pro` 项目文件
3. 配置构建套件（MinGW 或 MSVC）
4. 点击"运行"按钮编译并运行项目

### 或使用命令行编译

```bash
# 创建构建目录
mkdir build
cd build

# 运行qmake
qmake ../GraduatingDesice.pro

# 编译
mingw32-make  # 或 nmake（如果使用MSVC）

# 运行
.\debug\GraduatingDesice.exe  # 或 release\GraduatingDesice.exe
```

## 数据库

系统使用SQLite数据库，数据库文件名为 `huidian.db`，存储在程序运行目录下。

### 数据库表结构

系统包含以下数据表：
- **users**: 用户表（存储学生和管理员信息）
- **dormitories**: 宿舍表
- **electricity_records**: 电费使用记录表
- **recharge_records**: 充值记录表

## 安全特性

- 密码使用SHA256算法加密存储
- 充值和扣费操作使用事务保证数据一致性
- 支持角色权限管理（管理员/学生）

## 开发说明

### 核心类

1. **DatabaseManager**: 数据库管理类，单例模式，负责所有数据库操作
2. **StudentPanel**: 学生面板类，实现学生端功能
3. **AdminPanel**: 管理员面板类，实现管理员端功能
4. **loding_wd**: 登录窗口类，负责用户身份认证

## 常见问题

**Q: 数据库文件在哪里？**  
A: 数据库文件 `huidian.db` 会自动创建在程序运行的目录下。

**Q: 如何重置数据库？**  
A: 删除 `huidian.db` 文件，重新运行程序即可重新初始化数据库。

**Q: 忘记管理员密码怎么办？**  
A: 删除 `huidian.db` 文件，重新运行程序会创建默认管理员账号 admin/admin123。
